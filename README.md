# 프로젝트 소개
**프로젝트 이름** : NewsAPI

**프로젝트 설명**  
* 뉴스 조회 앱
* URLSession과 WebView를 사용하여 최신 뉴스 조회
* Realm을 통한 읽은 뉴스 목록 제공 


**앱 실행 최소 버전** : ios 15.0 이상
